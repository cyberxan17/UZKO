# UZKO

Tashqi platforma wrapperlari va deploy qoldiqlarisiz tozalangan loyiha.

## Local ishga tushirish

```bash
npm install
npm run dev
```

## Build tekshirish

```bash
npm run build
```

## Builddan keyin server preview

```bash
npm start
```

`node_modules` va `dist` `.gitignore`da turadi, GitHubga push qilinmaydi.
